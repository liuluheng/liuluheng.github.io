* Tools
** Visualization Toolkits
+ [[http://d3js.org/][D3]]  - A JavaScript library for data-driven DOM manipulation,
  interaction and animation. Includes utilities for visualization
  techniques and SVG generation.
+ [[https://processing.org/][Processing]] and [[http://processingjs.org/][Processing.js]]  - A popular Java-like graphics and
  interaction language and IDE. Processing has a strong user community
  with many examples.
+ [[http://mbostock.github.io/protovis/][Protovis]] - JavaScript visualization language, predecessor of d3.
+ HTML/!JavaScript/XML - use standard web technologies to build the
  visualization. You may use libraries such as [[http://jquery.com/][jQuery]], [[http://dojotoolkit.org/][Dojo]], and the
  [[https://developers.google.com/maps/][Google Maps API]] to help build your visualization.
+ [[http://polymaps.org/][PolyMaps]] – create map displays with JavaScript & SVG.
+ [[http://flare.prefuse.org/][Flare]] - Visualization toolkit for Adobe Flash.
+ [[http://modestmaps.com/][Modest Maps]] - Mapping library for Flash.
+ [[http://prefuse.org/][Prefuse]] - Visualization toolkit for Java.
+ [[http://www.cs.ou.edu/~weaver/improvise/][Improvise]] - Java system supporting coordinated views.
+ [[http://ivtk.sourceforge.net/][InfoVis Toolkit]] - A Java visualization library, from INRIA France.
+ [[http://www.cs.umd.edu/hcil/piccolo/learn/comparison.shtml][Piccolo]] - A Java library for zoomable UIs, from the University of
  Maryland (Java and .NET).
+ [[http://www.vtk.org/][VTK]] - A scientific visualization library (C++ with wrappers for
  other languages).
** Statistical Data Analysis Tools
+ [[http://www.tableausoftware.com/academic/students][Tableau for Student]] – get Tableau free license as a student.
+ [[http://www.tableausoftware.com/public/][Tableau Public]] - a free version of Tableau which publishes to the
  web.
+ [[http://ggplot2.org/][GGplot2]] - a graphics language for R.
+ [[http://www.ggobi.org/][GGobi]] - visualizations for multivariate data.
+ [[http://www.cs.ou.edu/~weaver/improvise/][Improvise]] - a visualization tool supporting a variety of
  visualization types.
+ [[http://www.mediavirus.org/parvis/][ParVis]] - software for parallel coordinates.
+ [[http://www.cs.umd.edu/hcil/timesearcher/][TimeSearcher]] - interface for time-series data from U Maryland.
+ [[http://www.cs.umd.edu/hcil/treemap/][TreeMap]] - tree-mapping software from U Maryland.

** Network Analysis Tools
+ [[http://nodexl.codeplex.com/][NodeXL]] graph analysis plug-in for Excel.
+ [[https://gephi.github.io/][Gephi]] - a graph analysis application.
+ [[http://graphexploration.cond.org/][GUESS]] - a combined visual/scripting interface for graph analysis.
+ [[http://pajek.imfm.si/doku.php][Pajek]] - another popular network analysis tool.
+ [[http://networkx.github.io/][NetworkX]] - graph analysis library for Python.
+ [[http://snap.stanford.edu/][SNAP]] - graph analysis library for C++.
