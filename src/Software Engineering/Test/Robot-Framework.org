* Overview
Homepage: http://code.google.com/p/robotframework/

Robot Framework is a keyword-driven test automation framework created
by Pekka Klärck at Nokia Siemens Networks in 2005. One of its early design goals was A-TDD
support. It was open-sourced in 2008 and is available at www.robotframework.org.

* External Supporting Tools
http://code.google.com/p/robotframework/wiki/SupportingTools#External_tools
** Robot IDE (RIDE)
[[http://code.google.com/p/robotframework-ride/][RobotIDE]] is a tool for editing test data.
** Manual Test Reporter (Mabot)
[[http://code.google.com/p/robotframework-mabot/][Mabot]] is a tool for manual test result reporting.
** Selenium IDE plugin
[[https://github.com/denschu/selenium-ide-format-robotframework][Selenium IDE plugin]] that supports copy/paste or export from Selenium
IDE into robot framework keywords
** Pybot wrapper script
[[https://github.com/jstaffans/robotbatch][Pybot wrapper script]] to run suites/tests in parallel by splitting them
up in batches
** Watir
[[https://github.com/semperos/watir-robot-gui][Watir specific gui]] for creating robot framework tests
